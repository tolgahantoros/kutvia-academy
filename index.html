Eğitim Programları

Yazılım Mühendisliği

Modern web geliştirme, API tasarımı, test kültürü ve sürüm yönetimi etrafında yapılandırılmış tam kapsamlı akış.

  • HTML, CSS ve JavaScript
  • React tabanlı arayüzler
  • API entegrasyonu
  • Test ve yayın süreçleri

Siber Güvenlik

Tehdit analizi, güvenli sistem tasarımı ve olay müdahale süreçlerine odaklanan pratik bir içerik seti.

  • Temel güvenlik mimarisi
  • Laboratuvar senaryoları

  • Risk farkındalığı

  • Karar destek sistemleri
  • Etik kullanım çerçevesi

Takım İçi Mentorluk
